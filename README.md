## Tech Stack

**Client:** React,CSS,HTML



## Features

- Create Notes
- Delete Notes
- Add description to notes
- Cross platform
- Responsive
- Saves all your notes in localStorage.


## Run Locally

Clone the project

```bash
  git clone https://github.com/0Pixel0/ikeep
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
